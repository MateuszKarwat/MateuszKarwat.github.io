New version of Napi

Napi with Mojave's Dark Mode support

I just released a new version of Napi. Version 1.1.0 introduces a full support for Mojave’s Dark Mode.

In addition, this version adds a completely new menu to the Status Bar with option to download subtitles using a specific subtitle provider.
